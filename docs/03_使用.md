# 03 使用

## 快速开始

1. 安装依赖（见 [02_开发.md](./02_开发.md)）。
2. 准备数据集，路径与格式与 `speedvqa/configs/default.yaml` 中 `data` 节点一致。
3. 复制或修改 **`speedvqa/configs/default.yaml`**（`dataset_path`、`batch`、轮数、骨干等）。

## 训练与预测

- **Python 模块**：`python -m speedvqa.cli.onekey_train`（及 `onekey_predict`、`onekey_export`）；已安装包时可用 **`speedvqa-train`** 等。
- **Shell**：仓库根 `scripts/onekey_train.sh` 等（内部调用上述模块）。

训练产出、日志等默认落在 **`runs/`** 下，约定见 **[04_部署.md](./04_部署.md)** 中「`runs/` 输出目录」一节。

## 配置要点（摘要）

- **`data.dataset_path`**：数据集根目录；其下可按实现放置 `annotations/`、`vqa_labels.jsonl` 等。
- **`data.answer_mapping`**：将多种肯定/否定表述映射到统一二分类标签。

完整键说明以 `default.yaml` 内注释与 **[01_设计.md](./01_设计.md)**（配置与数据要点、典型工作流）为准。

## 示例脚本

`speedvqa/examples/` 下含导出、基准、超参等可运行示例；具体文件名与用途可在该目录中浏览。

## 推理 API

日常使用 **`ROIInferencer`**（PyTorch / ONNX / TensorRT）详见 **`speedvqa/inference/README.md`**。
