# SpeedVQA 使用指南

本目录按 **大功能方向** 拆分为编号文档（`01_*.md` …），便于按角色与阶段查阅；模块级 API 说明见 `speedvqa/inference/README.md`、`speedvqa/export/README.md`。

## 建议阅读顺序

| 顺序 | 文档 | 适合场景 |
|------|------|----------|
| 1 | [01_设计.md](./01_设计.md) | 定位、架构、配置与数据设计、工作流、扩展骨干 |
| 2 | [02_开发.md](./02_开发.md) | 本地环境、推荐导入、测试与开发入口 |
| 3 | [03_使用.md](./03_使用.md) | 训练、预测、一键脚本与日常操作 |
| 4 | [04_部署.md](./04_部署.md) | 导出、推理、`runs/` 目录与清理 |

## 模块与示例（仓库内）

| 路径 | 内容 |
|------|------|
| `speedvqa/inference/README.md` | `ROIInferencer` API |
| `speedvqa/export/README.md` | `ModelExporter` 与导出格式 |

完整文档索引见仓库根目录 **[README.md](../README.md)**。
